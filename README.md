# oulankajoki-vr-tour
The Oulankajoki Supersite Virtual Tour Demo is an interactive, georeferenced VR application developed within the Digital Waters (DIWA) Flagship Program at the University of Oulu, Finland. Built using Cesium ion, Unity 2022.3 LTS, and the Meta XR SDK, the application streams a real-world LiDAR point cloud and satellite basemap of the Oulankajoki Supersite in Northern Finland directly from the cloud, enabling immersive on-foot and aerial exploration through a standalone Meta Quest 3 headset. The experience features six interactive markers displaying scientific data panels and site photography, three gaze-based teleport hotspots, and full six-degrees-of-freedom navigation using Quest 3 controllers — all without requiring any PC connection at runtime. This repository documents the complete development workflow through a step-by-step tutorial, a comprehensive 30-step linear process guide, and a live dashboard embedding both a WebGL browser version and a real-time Facebook Live stream of the VR session, making the Oulankajoki Supersite accessible to remote audiences worldwide through any standard web browser.
