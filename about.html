Repository Overview

OULANKAJOKI SUPERSITE
VIRTUAL TOUR DEMO

A fully standalone, georeferenced, interactive VR application built with Cesium ion and Unity for the Meta Quest 3 headset. Developed within the Digital Waters (DIWA) Flagship Program at the University of Oulu, Finland. The application streams a real-world point cloud and satellite basemap of the Oulankajoki Supersite directly from Cesium ion cloud, enabling immersive exploration on foot and from the air.

Technology stack
☁️
Cesium ion
Cloud platform for streaming georeferenced 3D point cloud and global satellite basemap. Free tier used throughout.
🎮
Unity 2022.3 LTS
Game engine used to build the VR scene, scripts, markers, billboards and interactions. Universal Render Pipeline.
🥽
Meta Quest 3
Standalone VR headset. Target deployment platform. App runs without any PC connection once installed.
📦
Meta XR SDK
All-in-One SDK providing OVR Camera Rig, controller input, hand tracking and Quest 3 optimisation tools.
🌐
WebGL
Browser-based build of the VR scene allowing mouse and keyboard navigation without any installation.
💻
C# Scripts
Eight custom scripts: VRMovement, ControllerRay, SpotMarker, Slideshow, BillboardButton, LookAtPlayer, TeleportSpot, SetPlayerLocation.
Application features
🌍
Georeferenced Point Cloud
Real LAZ scan of Oulankajoki streamed from Cesium ion with satellite basemap underneath.
🕹
Free VR Navigation
Left thumbstick moves, right thumbstick rotates. A button flies up, B button descends for bird's eye view.
⚪
Interactive Markers
6 clickable sphere markers — 1 data panel with 7-slide slideshow and 5 site photo billboards.
➡️
Ray Interactors
White rays from both Quest 3 controllers detect and interact with all markers and buttons.
🟡
Teleport Hotspots
3 yellow cylinder hotspots — gaze for 2 seconds to teleport instantly to that location.
📺
Facebook Live Stream
Stream directly from Quest 3 to Facebook Live and embed on any website for remote audiences.
🌐
WebGL Browser Version
Full scene accessible in any browser via URL — no download or installation required.
✈️
Portable for Vienna
Only Quest 3 headset and WiFi needed. Runs standalone anywhere in the world.
About this project: The Oulankajoki Supersite Virtual Tour Demo was developed as part of the Digital Waters (DIWA) Flagship Program at the University of Oulu, Finland. It demonstrates the potential of low-cost, open-standard geospatial tools for creating immersive scientific visualisations accessible to a wide audience through consumer VR hardware and standard web browsers.
